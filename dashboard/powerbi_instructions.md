# Power BI Dashboard – Setup Guide

## Overview

This guide walks you through building an interactive **Credit Risk Scorecard Dashboard** in Power BI Desktop using the scored output generated by the Python pipeline.

---

## Prerequisites

| Item | Details |
|------|---------|
| **Software** | Power BI Desktop (free download from [Microsoft](https://powerbi.microsoft.com/desktop/)) |
| **Data File** | `outputs/scorecard_output.csv` — generated by running `python main.py` |
| **Optional** | `outputs/iv_summary.csv` for an Information Value chart |

---

## Step 1 – Import Data

1. Open **Power BI Desktop**.
2. Click **Home → Get Data → Text/CSV**.
3. Navigate to `outputs/scorecard_output.csv` and click **Open**.
4. In the preview dialog, verify:
   - Delimiter = **Comma**
   - Data types are auto-detected correctly (numeric columns should show *Decimal Number* or *Whole Number*).
5. Click **Load**.
6. Repeat for `outputs/iv_summary.csv` (optional).

---

## Step 2 – Data Preparation (Power Query)

1. Click **Transform Data** to open Power Query Editor.
2. Rename the query to `ScoredLoans`.
3. Verify column types:
   - `credit_score` → **Whole Number**
   - `predicted_proba` → **Decimal Number**
   - `bad_loan` → **Whole Number**
   - `risk_tier` → **Text**
4. Create a **Score Band** column:
   - **Add Column → Custom Column**:
     ```
     if [credit_score] >= 800 then "800-900"
     else if [credit_score] >= 720 then "720-799"
     else if [credit_score] >= 660 then "660-719"
     else if [credit_score] >= 600 then "600-659"
     else "300-599"
     ```
5. Click **Close & Apply**.

---

## Step 3 – Create Measures (DAX)

Add these DAX measures to the `ScoredLoans` table:

```dax
-- Total Loans
Total Loans = COUNTROWS(ScoredLoans)

-- Default Rate
Default Rate = DIVIDE(
    CALCULATE(COUNTROWS(ScoredLoans), ScoredLoans[bad_loan] = 1),
    COUNTROWS(ScoredLoans),
    0
)

-- Approval Rate (score >= 660)
Approval Rate = DIVIDE(
    CALCULATE(COUNTROWS(ScoredLoans), ScoredLoans[credit_score] >= 660),
    COUNTROWS(ScoredLoans),
    0
)

-- Average Credit Score
Avg Credit Score = AVERAGE(ScoredLoans[credit_score])

-- Average Default Probability
Avg Default Prob = AVERAGE(ScoredLoans[predicted_proba])
```

## Step 4 – Build Dashboard Pages

### Page 1: Executive Summary

| Visual | Field / Measure | Notes |
|--------|-----------------|-------|
| **Card** | `Total Loans` | Large font, top-left |
| **Card** | `Default Rate` | Format as percentage |
| **Card** | `Avg Credit Score` | |
| **Card** | `Approval Rate` | Format as percentage |
| **Donut Chart** | Legend = `risk_tier`, Values = Count of `loan_id` | Show portfolio concentration |
| **Histogram** | X = `credit_score` (binned), Y = Count | 20-point bins |

### Page 2: Risk Tier Deep Dive

| Visual | Field / Measure | Notes |
|--------|-----------------|-------|
| **Stacked Bar** | Axis = `risk_tier`, Values = Count, Legend = `bad_loan` | Shows defaults per tier |
| **Table** | Rows = `risk_tier`, Values = Count / Default Rate / Avg Score | Summary table |
| **Line Chart** | X = `Score Band`, Y = `Default Rate` | Monotonic relationship expected |

### Page 3: Feature Importance (Optional)

| Visual | Field / Measure | Notes |
|--------|-----------------|-------|
| **Bar Chart** | Axis = `feature` (from IV table), Values = `iv` | Horizontal, sorted desc |
| **Text Box** | "Features with IV > 0.10 are considered strong predictors" | Annotation |

---

## Step 5 – Formatting & Theming

1. **Theme**: Use a dark theme or the built-in *Executive* theme.
2. **Colors**:
   - Super Prime → Dark Green `#27ae60`
   - Prime → Light Green `#2ecc71`
   - Near Prime → Yellow `#f1c40f`
   - Subprime → Orange `#e67e22`
   - Deep Subprime → Red `#e74c3c`
3. **Fonts**: Segoe UI or Calibri, 12pt body / 18pt headers.
4. Add the project title: **"Credit Risk Scorecard Dashboard"** as a text box at the top.

---

## Step 6 – Add Slicers (Filters)

Add these slicers to each page for interactivity:

- **Risk Tier** (dropdown)
- **Grade** (multi-select)
- **Home Ownership** (dropdown)
- **Term** (36 / 60 months)

---

## Step 7 – Publish (Optional)

1. **File → Publish → Publish to Power BI Service**.
2. Set up a **scheduled refresh** if connected to a live data source.
3. Share the dashboard link with stakeholders.

---

## Troubleshooting

| Issue | Solution |
|-------|----------|
| CSV not loading | Ensure the pipeline has been run (`python main.py`) and `scorecard_output.csv` exists |
| Columns show as text | Manually set data types in Power Query (Step 2) |
| Donut chart shows one slice | Check that `risk_tier` column has multiple distinct values |
| Measures return blank | Verify measure formulas reference the correct table name |
